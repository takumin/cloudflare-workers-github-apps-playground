# cloudflare-workers-github-apps-playground

Cloudflare Workers GitHub Apps Playground

## Setup

To install dependencies:

```bash
brew install aquaproj/aqua/aqua
aqua install
pnpm install
```

## Running

To run:

```bash
pnpm run dev
```
